# 03 — Files & Modules (061–080)

## Purpose
Learn to structure multi-file programs and manage external data.

## Topics Covered
- Reading and writing files
- CSV and JSON handling
- Modules and imports
- Virtual environments
- Logging
- Command-line arguments

## Objective
Build multi-file applications with clean separation of concerns.

## Completion Criteria
- Create at least one multi-file CLI project
- Use virtual environments correctly
- Manage dependencies with requirements.txt
